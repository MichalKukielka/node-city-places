1) Zainstaluj node.js ze strony: https://nodejs.org/en/download/
2) Przejdź w terminalu systemowym do katalogu głównego aplikacji
3) Wywołaj polecenie npm install —production
4) Uruchom serwer poleceniem node app.js
5) Komunikat potwierdzający poprawne uruchomienie serwera to: Course project app has started!
6) Aplikacja dostępna jest lokalnie na porcie 3000 pod adresem http://127.0.0.1:3000/
